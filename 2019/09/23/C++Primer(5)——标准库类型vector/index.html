C++Primer(5)——标准库类型vector

发表于 2019-09-23 | 分类于 C++Primer | 本文总阅读量 次
字数统计: 1.2k 字 | 阅读时长 ≈ 4 分钟

标准库类型vector表示对象的集合，其中所有对象的类型都相同，也常被称作容器。vector是一个类模板，模板本身不是类或者函数。vector是模板而非类型，由vector生成的类类型必须包含vector中元素的类型，比如vector<int>。

列表初始化对象

用花括号括起来的0个或者多个初始元素值被赋给vector对象。

注意：
1.在使用拷贝初始化时只能提供一个初始值。
2.如果提供的是一个类内初始值则只能使用拷贝初始化或者花括号的形式初始化。
3.如果提供的初始元素值得列表，则只能把初始值都放在花括号内进行列表初始化，而不能放在圆括号里。

值初始化对象

可以只提供vector对象容纳得元素数量而忽略初始值，这个初始值由vector对象中元素的类型决定。

如果只提供了元素个数而没有初始值，则只能用直接初始化。
